# Octane Ajax Market Research Notes

## Etsy Fee Structure (2026)
- $0.20 listing fee per item (renews every 4 months or upon sale)
- 6.5% transaction fee on total sale price (including shipping)
- 3% + $0.25 payment processing fee (US)
- **Total take for a $10 digital product: ~$0.20 + $0.65 + $0.55 = $1.40 (~14%)**
- **Total take for a $5 digital product: ~$0.20 + $0.325 + $0.40 = $0.925 (~18.5%)**
- Digital products pay the same fees as physical items
- Etsy Ads (optional): 12-15% of attributed sale if using Offsite Ads
- Source: https://nifty.ai/post/how-much-does-etsy-take-per-sale

## Etsy AI Policy (2026)
- Sellers MUST disclose AI use within listing description (tightened June 10, 2025)
- AI-assisted products must be listed under "Designed by" not "Made by" or "Handmade"
- Listings that skip disclosure get flagged and removed by automated moderation
- Bulk-uploading unchecked AI images triggers shop-wide warnings
- AI cannot manufacture social proof or reviews
- Sellers must maintain documentation of creative process
- Selling prompt bundles as standalone products is prohibited
- Source: https://www.etsy.com/legal/creativity/, https://ecombalance.com/ai-content-policies-2026/

## Etsy Digital Product Economics
- To hit $2,000/month profit at $5 products: need ~450-500 sales = 40k-60k visits at 2% conversion
- One seller made $93K revenue selling digital downloads
- Digital products = zero fulfillment cost, zero shipping, infinite inventory
- Main cost is creation time (which AI can dramatically reduce)

## Etsy Open API v3 - Listing Automation Capabilities
Source: https://developer.etsy.com/documentation/tutorials/listings

**Key Findings:**
1. Etsy API v3 fully supports programmatic listing creation via `createDraftListing`
2. Digital products are supported - set `type` parameter to "download"
3. Digital files uploaded via `uploadListingFile` endpoint
4. Listing lifecycle: draft → published → deactivated/sold_out/expired
5. Required OAuth scopes: `listings_r`, `listings_w` (and `listings_d` for deletion)
6. Required fields for createDraftListing: quantity, title, description, price, who_made, when_made, taxonomy_id
7. Images uploaded separately via `uploadListingImage`
8. **Critical:** Listings can be created as DRAFTS first, then published separately - this supports the Review Gate model perfectly

**API Access Requirements:**
- Need to register an Etsy App at developers.etsy.com
- Get API keystring and shared secret
- OAuth 2.0 authentication flow
- Free to use (no API fees beyond normal Etsy seller fees)

**This confirms Octane Ajax can fully automate the Etsy listing pipeline:**
Nova researches → Forge generates listing content → Review Gate (human approves) → API creates draft listing → Human publishes on Etsy

## Design Automation Options for Octane Ajax

### Option A: Canva Connect API
- Canva has a full Connect API (canva.dev/docs/connect/)
- Can programmatically create designs from brand templates
- "Autofill" feature: fill template placeholders with dynamic content
- Can export finished designs as images/PDFs
- Requires Canva Pro ($13/mo) + API access
- Best for: template-based products (planners, trackers, workbooks)

### Option B: Pure Code Generation (PDF/SVG)
- Use Python libraries (ReportLab, FPDF2, WeasyPrint) to generate PDFs programmatically
- Full control, zero external dependency cost
- Can create: planners, trackers, checklists, workbooks
- Harder to make "beautiful" without design expertise
- Best for: utility-first products where function > aesthetics

### Option C: AI Image Generation + PDF Assembly
- Use GPT Image / DALL-E for decorative elements, covers, illustrations
- Use Python to assemble into multi-page PDFs
- Hybrid approach: AI for beauty, code for structure
- Cost: $0.04-$0.17 per image

### Recommended Approach for Octane Ajax:
**Start with Option B (code-generated PDFs) for utility products, enhanced with Option C (AI images for covers/decorative elements).** This keeps costs near zero for the actual product creation, with AI costs only for listing images and cover art. Migrate to Canva API integration later for more visually complex products.

## LLM Integration Architecture for Agents

### Nova Agent (Trend Scout / Idea Generator)
- **Input:** Reddit API data, Etsy trending searches, Google Trends
- **LLM Task:** Analyze trends, identify gaps, generate product ideas with market sizing
- **Model:** GPT-4o-mini ($0.15/$0.60 per 1M tokens)
- **Estimated cost per cycle:** ~$0.001 (500 tokens in, 1000 out)
- **Data sources to integrate:**
  - Reddit API (free, 100 requests/min)
  - Etsy trending searches (scraping or API)
  - Google Trends (unofficial API via pytrends)

### Forge Agent (Product Creator)
- **Input:** Nova's product idea + niche research
- **LLM Task:** Generate listing title, description, tags, pricing recommendation
- **Model:** GPT-4o-mini for text, GPT Image Mini for cover art
- **Estimated cost per cycle:** ~$0.04-$0.05 (text + 1 image)
- **Output:** Complete listing package ready for Review Gate

### Pixel Agent (Marketing / SEO)
- **Input:** Forge's listing + platform requirements
- **LLM Task:** Optimize SEO tags, generate social media copy, A/B test titles
- **Model:** GPT-4o-mini
- **Estimated cost per cycle:** ~$0.001
- **Output:** Optimized listing + marketing materials

### Total Cost Per Product Pipeline:
- Nova: $0.001
- Forge: $0.05
- Pixel: $0.001
- **Total: ~$0.05 per product**
- **$10/month API budget = ~200 products/month**
- This is MORE than enough for a solo operation
